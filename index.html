difference between "==" and "==="

"==" :- the "==" performs loose equality comparizon that does type coercion if necessary for comparizon

            2=="2"  
            output
            true

"===" :- the "===" performs strict equality comparizon that does not do type coercion it compares both operand and its type

            2==="2"
            output
            false

what is temporaldead zone(TDZ)

A let or const variable is said to be in a "temporal dead zone" (TDZ) from the start of the block until code
execution reaches the line where the variable is declared and initialized. While inside the TDZ, the
variable has not been initialized with a value, and any attempt to access it will result in a ReferenceError

                console.log(a); //a is in temporal dead zone

what is meant by first class function?

functions in called as the first class function when they are treated like other variables. So the functions can be
assigned to any other variable or passed as an argument or can be returned by another function. 

            function sayHello(name){
